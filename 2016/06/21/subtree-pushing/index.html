Subtree Pushing for GitHub Pages

This repository has a source branch which is the default branch now.
When we run hexo generate it builds the site in the public subfolder.

Making sure public folder is not in .gitignore, we commit and do a subtree push.
git subtree push --prefix public origin master

This will push the contents of the public folder on the source branch to the root of the master branch.

Similarly, for project pages we can do the same thing with master/gh-pages branches instead of source/master respectively.
